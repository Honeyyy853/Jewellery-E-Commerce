## 📌 Project Overview
The core customer-facing interface. Featuring advanced product discovery workflows, dynamic carts, and rigorous Razorpay test integration, this application handles all UI/UX logic separating presentation from backend complexity.

## 🚀 Key Features
- **Modern User Interface:** Sleek, responsive, and intuitive browsing experience catering to premium aesthetics.
- **Comprehensive Product Catalog:** Smooth filtering capabilities and extensive product views.
- **Frictionless Cart & Checkout:** Persistent, session-managed cart logic guiding the user into payment gateways setup.
- **Razorpay Integration:** Seamless test transactions demonstrating real-world e-commerce viability.
- **Authentication Routes:** Safe user logins and profile management handling personalized data gracefully.

## 💻 Technology Stack
- **Library:** React.js
- **Build Tool:** Vite 
- **Payment Gateway:** Razorpay Standard Checkout SDK
- **Styling Architecture:** Custom CSS frameworks optimized for fast paint times.
- **API Networking:** Mapped to RESTful PHP resources via Axios/Fetch.

## 🛠️ Setup & Run Instructions

### Prerequisites
- Node.js installed environment.
- Running PHP/MySQL `Jewellerydb` API.

### Installation Steps
1. Target the correct user directory:
   ```bash
   cd Jewellery-User-Panel-main
   ```
2. Build the application dependencies:
   ```bash
   npm install
   ```
3. Execute the server instance:
   ```bash
   npm run dev
   ```
4. Application typically hosts concurrently at `http://localhost:5174` (or `5173` depending on ports).
